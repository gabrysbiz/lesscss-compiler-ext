Extended LessCSS Compiler

Extended version of the LessCSS Compiler (Java library which compiles Less source files to the CSS code)

View on GitHub Release 1.0 docs LATEST (version under development) docs

About

Extended version of the LessCSS Compiler (Java library which compiles Less source files to the CSS code). Adds features such as:

  • caching (see caching data):
    • compiled code
    • source code
  • supports sources located at:
    • local drives
    • network, protocols: HTTP, HTTPS
    • custom - defined by programmers

Compatibility

The compatibility with the Less language is determined by:

  • native compiler implementation (biz.gabrys.lesscss.compiler.LessCompiler)
  • supported locations of sources (see supported locations matrix)

Requirements

The compiler to run requires:

  • Java 5.0 or higher
  • Third-Party Dependencies (see list)

Download

You can download the compiler from docs pages or using various dependency management tools:

  • Apache Maven
  • Apache Buildr
  • Apache Ivy
  • Groovy Grape
  • Gradle
  • Grails
  • Scala SBT
  • Leiningen

Example for Maven:

<project>
   ...
   <dependencies>
      <dependency>
         <groupId>biz.gabrys.lesscss</groupId>
         <artifactId>extended-compiler</artifactId>
         <version>1.0</version>
      </dependency>
      ...
   <dependencies>
   ...
</project>
